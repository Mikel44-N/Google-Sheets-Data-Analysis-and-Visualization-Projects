# Google-Sheets-Data-Analysis-and-Visualization-Projects
This is a hands-on Google Sheets project covering spreadsheet fundamentals, formulas and functions, pivot tables, and data visualization.
The work simulates real-world business scenarios and demonstrates how Google Sheets can be used to organize data, perform analysis, and communicate insights through interactive dashboards and charts.

Spreadsheet Fundamentals (HR Data Management)
Business Scenario
Role: HR Administrator at ACME Corporation
Dataset includes employee job titles, departments, salaries, performance scores, and satisfaction data.

Objective
Organize and structure HR data to improve readability, data entry accuracy, and employee monitoring.

Tasks Completed
Applied currency formatting to salary fields and custom date formatting to start and exit dates
Formatted employee ID and supervisor ID fields as text
Used custom number formatting to standardize phone numbers
Implemented data validation to restrict age entries to 18+
Added email validation to flag invalid email addresses
Applied color scale conditional formatting to highlight salary distribution
Highlighted employees over age 60 using conditional formatting
Used checkbox-based logic to identify active vs inactive employees
Created and saved filter views to monitor active employees by performance and salary thresholds
Built dropdown menus for performance ratings using data validation

Outcome
A clean, controlled HR dataset that supports accurate data entry, filtering, and employee monitoring.


Formulas & Functions (Sales & Orders Analysis)
Business Scenario
Role: Data Analyst at Northwind Traders
Analyzed order, product, customer, and discount data to support management reporting.

Objective
Manipulate and analyze transactional data to generate insights and summary metrics.

Tasks Completed
Calculated delivery windows using date arithmetic
Used COUNTUNIQUE and COUNTBLANK to analyze customer and order activity
Removed duplicates and sorted country-level data
Calculated gross revenue using derived fields
Applied SUMIFS, AVERAGEIFS, and COUNTUNIQUEIFS for conditional analysis
Built nested IF logic to calculate discounts and delivery status (on-time, late, at-risk)
Used logical functions (IF, AND, OR) to define priority customers
Applied IFERROR to handle division errors in growth analysis
Used XLOOKUP for exact and approximate matches across tables
Created date-based fields (Year, Week, Months Discontinued)
Used text functions (CONCATENATE, TRIM, LEFT, PROPER, SPLIT) for data cleaning
Leveraged Smart Fill for efficient data completion

Outcome
A formula-driven analytical dataset that updates dynamically and supports business decision-making.


Pivot Tables (Marketing Campaign Analysis)
Business Scenario

Role: Marketing Analyst at Maven Marketers
Analyzed paid search campaign performance for a U.S. shopping mall.

Objective

Explore campaign performance, identify optimization opportunities, and generate insights for future campaigns.

Tasks Completed
Built pivot tables to analyze impressions, clicks, conversions, and costs
Calculated campaign-level metrics by platform and match type
Used value settings to show percentages of rows and columns
Grouped keywords into meaningful categories (Sales, Codes, Other)
Added grouped fields back into the raw dataset
Applied numeric grouping to campaign and impression ranges
Created calculated fields for cost per click and conversion rate
Added slicers for platform and date filtering
Sorted and filtered high-performing keywords

Outcome
Interactive pivot-based analysis enabling quick performance comparisons and optimization insights.
